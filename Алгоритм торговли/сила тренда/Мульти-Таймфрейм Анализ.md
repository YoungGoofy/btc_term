**Почему это важно:**

1-минутный UP тренд внутри 15-минутного нисходящего тренда — это коррекция, не разворот. Анализ нескольких таймфреймов защищает от ложных сигналов.

**Четыре таймфрейма:**

|Таймфрейм|Роль|Что смотрим|
|---|---|---|
|1m|Тактический|3-candle сигнал, тайминг входа|
|5m|Подтверждение|Направление последних 3 свечей|
|15m|Структура|Контекст предыдущей 15m свечи|
|4h|Макро|Общее направление рынка|

**Правило согласованности:**

Входить только когда минимум 3 из 4 таймфреймов согласованы.

**Пример — сильный UP:**

1m: 3 зелёные ✅ | 5m: восходящий ✅ | 15m: предыдущая свеча зелёная ✅ | 4h: бычий ✅

→ 4/4 → максимальный размер позиции

**Пример — конфликт:**

1m: 3 зелёные ✅ | 5m: восходящий ✅ | 15m: нейтральная | 4h: нисходящий

→ 2/4 → пропуск или половинный размер

**Чтение предыдущей 15m свечи:**

Большая зелёная 15m + текущий 1m UP = усиление сигнала

Большая красная 15m + текущий 1m UP = возможный отскок, осторожно
